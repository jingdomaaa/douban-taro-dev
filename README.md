# douban-taro
##### 注意事项
###### 这里的接口请求不可以使用fetch,要使用Taro自己封装的request
###### 这个例子使用的是豆瓣的api,豆瓣目前开放的都是get的接口请求，

这里进行消息头设置由  Accept: 'application/json'修改成  Accept: 'json',
